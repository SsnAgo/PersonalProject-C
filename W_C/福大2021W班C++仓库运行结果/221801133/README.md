# 我的项目WordCount
---
## 运行方式
* 通过在命令行窗口(cmd)中输入：（其中第二、三个参数分别是输入、输出文件）
```c++
WordCount.exe input.txt output.txt
```
* 也可以通过编译器命令行输入参数来执行；

## 功能
本项目包括以下功能：oun
1. 统计文件的文本字符数；
2. 统计文本的单词数；
3. 统计文本的有效行数；
3. 统计最多的10个单词及其词频。
