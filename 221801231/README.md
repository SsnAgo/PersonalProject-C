# 项目简介
## 项目组成
wordcount.cpp
lib.h
lib.cpp
## 如何运行？
在visual studio2017下生成wordcount.exe后，在命令行下转入wordcount.exe所在的目录，input.txt和output.txt文件必须也在同一目录下
将wordcount.exe input.txt output.txt 作为命令行的三个参数输入即可

## 功能简介
使用该程序进行对文件的字符数，单词数，行数，以及出现最多次的单词数进行统计。
