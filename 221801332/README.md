#WordCount

##运行环境

>windows 10系统

>编译器：Visual Studio 2019

##使用方法

>命令行进入到根目录下

>执行指令：WordCount.exe input.txt result.txt ,其中输入、输出文件的路径可自行修改。

##实现功能

>统计文本字符数（默认输入仅有ascii字符）

>统计文本有效行数（包含非空白字符的行即是有效行）

>统计单词数（由4个字母开头，后跟0个或多个字母数字的字符串就是单词）

>计算出现频率最高的前十个单词（单词相同按字典序升序
